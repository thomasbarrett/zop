zopa simple vector processing library

Installation

Since this is a header only library, installation is very easy! Simply drag the header files into your project and include them. In the future, there may be a more complicated installation process, especially is hardware acceleration support is added via openCL or CUDA, but for now, we will keep it simple.

Example

#include <zop/Vector.h>
#include 

int main(void) {
    zop::Vector u{1, 2, 3};
    zop::Vector v{3, 1, 2};
    zop::Vector w = zop::Vector::cross(u, v);
    std::cout << w << std::endl;
}
