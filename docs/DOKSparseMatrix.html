DOKSparseMatrix

The DOKSparseMatrix class defines a 'dictionary-of-keys' sparse matrix implementation. The DOKSparseMatrix implementation is especially useful when constructing matrices since it has quick insertion of elements. Later, it can be converted to a more optimized structure such as a CSRSparseMatrix which is optimized for matrix-vector multiplication.
