zopa simple vector processing library

Vector

The Vector class contains data structures and utilities for working with n-dimension real vectors. Currently, only real numbers are supported, but soon complex numbers will also be supported.,

Class Functions

The following functions perform elementwise vector-vector operations.

static Vector add(const Vector &a, const Vector &b)
static Vector subtract(const Vector &a, const Vector &b)
static Vector multiply(const Vector &a, const Vector &b)
static Vector divide(const Vector &a, const Vector &b)

The following functions perform elementwise vector-double operations.

static Vector add(const Vector &a, double b)
static Vector subtract(const Vector &a, double b)
static Vector multiply(const Vector &a, double b)
static Vector divide(const Vector &a, double b)
static double dot(const Vector &a, const Vector &b) 
static Vector cross(const Vector &a, const Vector &b)
